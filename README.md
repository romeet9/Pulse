# Pulse ⚡️

**Pulse** is a beautiful, native macOS menu bar utility for monitoring and managing system processes. It helps you identify resource hogs and keeps your Mac running smoothly with a single click.

## Features

- **Liquid Glass UI**: A stunning, translucent interface that blends perfectly with macOS.
- **Smart Clean**: Intellectually identifies idle, memory-hungry background apps.
- **Visual Intelligence**: Features a native "Liquid Pulse" animation during system analysis.
- **App Manager**: Easily force quit or completely uninstall applications directly from the menu bar.
- **Native Performance**: Built with Swift and SwiftUI for maximum efficiency and negligible footprint.

## Installation

### Option 1: Build from Source
1. Clone this repository.
2. Open Terminal in the project folder.
3. Run:
   ```bash
   ./install_pulse.sh
   ```

### Option 2: Pre-built Release (For Office/Colleagues)
1. Download `Pulse.zip` from the Releases page.
2. Unzip the file.
3. Drag `Pulse.app` to your **Applications** folder.
4. **Important**: Since this app is not signed by an Apple Developer ID, you may see a warning ("Unverified Developer").
   - **Right-click (Control-click)** on Pulse.app and select **Open**.
   - Click **Open** in the dialog box.
   - You only need to do this once.

## Compatibility
- macOS 14.0 (Sonoma) or later.
